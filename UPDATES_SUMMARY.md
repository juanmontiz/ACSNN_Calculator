# ACSNN Calculator — Changelog

## v1.0.0 (Current)

First release. Pure NumPy neural network for atomic ionization cross-sections.

- CLI (`acsnn_cli.py`) and Tkinter GUI (`acsnn_gui.py`)
- Three theories: CDW-EIS, CTMC, Semiempiric (Rudd 1985)
- 20 automated tests
- Windows/Linux/Mac support
- CITATION.cff for Zenodo
